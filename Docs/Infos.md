## Rendering-methode : PictureBox

- Mit einer PictureBox können texturen und Sprites geladen und genutzt werden  

  Es ermöglicht uns eigene Sprites/Texturen/Bilder zu verwenden, um dinge wie  
  Den Start oder das Ende in speziellen symbolen darzustellen

- Mehr über die PictureBox unter [Microsoft.learn](https://learn.microsoft.com/de-de/dotnet/api/system.windows.forms.picturebox?view=windowsdesktop-8.0)

  // So ladet ihr ein Bild in eine PictureBox:  
  pictureBox1.Image = Image.FromFile("Bild & Pfad");
